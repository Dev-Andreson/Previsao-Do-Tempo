Nome do projeto: ViTemp
Resumo: O ViTemp é um site de previsão do tempo simples moderno e responsivo.

Funcionalidades Principais :
    Busca por cidade.
    Exibição de temperatura, condições climáticas (sol, chuva, etc.), umidade e velocidade do vento.
    Previsão para os próximos dias.
    Design responsivo (funciona bem no celular e desktop).

Tecnologias utilizadas:
    Front-end: Html, Css, JavaScript.
    Api: Weatherapi.
    Hospedagem: GitHub
